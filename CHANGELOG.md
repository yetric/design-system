# Changelog

## [0.5.0] - 2026-05-29

### ✨ Features

- Add `EmptyState` component — icon, title, description, action button
- Add `ConfirmDialog` component — AlertDialog wrapper with confirm/cancel callbacks
- Add `DatePicker` component — single date selection via react-day-picker v10
- Add `DateRangePicker` component — date range selection via react-day-picker v10
- Add `FileUpload` component — drag-and-drop with size/count validation and previews
- Add `Image` component — fallback, blur, aspect-ratio, lazy load
- Add `ColorPicker` component — HSL sliders, hex input, swatches
- Add `Charts` — `LineChart`, `BarChart`, `AreaChart`, `PieChart` wrappers via recharts
- Add programmatic `toast` API wrapping sonner
