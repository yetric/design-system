`@yetric/ui` ships a complete set of AI-native UI components. They are **provider-agnostic** — they do not call any AI API directly. You supply the stream, they handle the UI.

## useChat hook

The `useChat` hook wires the components together and handles message state, streaming, and error handling.

```tsx
import { AIChat, useChat } from "@yetric/ui";

export function Assistant() {
  const { messages, send, isLoading } = useChat({
    onRequest: async (messages) => {
      const res = await fetch("/api/chat", {
        method: "POST",
        body: JSON.stringify({ messages }),
      });
      // Convert the response body into an AsyncIterable<string>
      return streamToAsyncIterable(res.body!);
    },
  });

  return (
    <AIChat
      messages={messages}
      isLoading={isLoading}
      onSend={send}
    />
  );
}

async function* streamToAsyncIterable(stream: ReadableStream<Uint8Array>) {
  const reader = stream.pipeThrough(new TextDecoderStream()).getReader();
  try {
    while (true) {
      const { done, value } = await reader.read();
      if (done) break;
      yield value;
    }
  } finally {
    reader.releaseLock();
  }
}
```

## Component reference

### AIChat

A full chat interface. Manages scroll-to-bottom, empty state, suggestion chips, and the prompt composer internally.

```tsx
<AIChat
  title="Assistant"
  messages={messages}
  isLoading={isLoading}
  onSend={send}
  suggestions={["Summarise this", "Explain simply"]}
  onSuggestionSelect={send}
  height={600}
/>
```

### AIMessage

A single message bubble. Use `markdown` on assistant messages to render formatted responses.

```tsx
<AIMessage
  role="assistant"
  content="Here is a **formatted** response with `code`."
  markdown
  timestamp={new Date()}
/>
```

### PromptInput

A textarea that auto-resizes and submits on Enter (Shift+Enter for newline).

```tsx
<PromptInput
  value={value}
  onChange={setValue}
  onSubmit={handleSubmit}
  loading={isLoading}
  placeholder="Ask anything…"
/>
```

### StreamingText

Renders an `AsyncIterable<string>` token stream as it arrives.

```tsx
<StreamingText stream={tokenStream} cursor />
```

Or animates a pre-baked string character-by-character:

```tsx
<StreamingText text="Hello!" speed={30} />
```

### ThinkingIndicator

```tsx
<ThinkingIndicator variant="dots" size="md" label="Thinking…" />
```

Variants: `dots` | `pulse` | `spinner`. Sizes: `sm` | `md` | `lg`.

### ModelSelector

```tsx
<ModelSelector
  models={[
    { id: "gpt-4o", name: "GPT-4o", provider: "OpenAI" },
    { id: "claude-sonnet-4", name: "Claude Sonnet 4", provider: "Anthropic" },
  ]}
  value={selectedModel}
  onChange={setSelectedModel}
/>
```

### SuggestionChips

```tsx
<SuggestionChips
  chips={[
    { id: "1", label: "Summarise this" },
    { id: "2", label: "Explain simply" },
  ]}
  onSelect={(chip) => send(chip.label)}
  maxVisible={4}
/>
```
