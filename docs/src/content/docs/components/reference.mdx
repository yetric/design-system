## Component categories

| Category | Components |
|----------|-----------|
| **Actions** | Button, Toggle, ToggleGroup |
| **Forms** | Input, Textarea, Select, Checkbox, RadioGroup, Switch, Slider, Label, Form, DatePicker, DateRangePicker, ColorPicker, FileUpload, PinInput, TagsInput, MultiSelect, PhoneInput, NumberInput, Rating |
| **Feedback** | Alert, Badge, Progress, Skeleton, Toast, ThinkingIndicator |
| **Layout** | Card, Separator, ScrollArea, Stack, Grid, Box, Paper, AspectRatio |
| **Navigation** | Breadcrumb, Pagination, Tabs, Stepper, Menubar, NavigationMenu, NavItem |
| **Overlays** | Dialog, AlertDialog, Popover, Tooltip, DropdownMenu, ContextMenu, HoverCard, Drawer, Command, CommandPalette |
| **Data** | Table, DataGrid, Charts |
| **Media** | Image, Avatar, VideoPlayer, MapView |
| **Typography** | Text, Blockquote, Code, Kbd |
| **Utility** | Accordion, Collapsible, Timeline, EmptyState, Indicator, Loader, Tour, AuthForms |
| **AI** | AIChat, AIMessage, PromptInput, StreamingText, ThinkingIndicator, ModelSelector, SuggestionChips |

## All components are exported from the package root

```ts
import {
  Button,
  AIChat,
  DataGrid,
  // ...
} from "@yetric/ui";
```

No deep imports needed.
