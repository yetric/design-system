## Requirements

- React 18 or later
- Node.js 18 or later
- Tailwind CSS 4 (for customising the theme)

## Install

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @yetric/ui
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @yetric/ui
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @yetric/ui
    ```
  </TabItem>
</Tabs>

## Import styles

Import the stylesheet once at the root of your app:

```tsx
import "@yetric/ui/styles";
```

This loads the component styles and the default color theme.

## Use a component

```tsx
import { Button } from "@yetric/ui";

export function Example() {
  return <Button variant="primary">Get started</Button>;
}
```
