import { Steps } from "@astrojs/starlight/components";

<Steps>

1. **Create a Vite app** (skip if you have one):

   ```bash
   npm create vite@latest my-app -- --template react-ts
   cd my-app
   npm install
   ```

2. **Install @yetric/ui**:

   ```bash
   npm install @yetric/ui
   ```

3. **Import the stylesheet** in your entry point:

   ```tsx
   // src/main.tsx
   import { StrictMode } from "react";
   import { createRoot } from "react-dom/client";
   import "@yetric/ui/styles";
   import App from "./App";

   createRoot(document.getElementById("root")!).render(
     <StrictMode>
       <App />
     </StrictMode>
   );
   ```

4. **Use components**:

   ```tsx
   // src/App.tsx
   import { Button, Card, CardContent, CardHeader, CardTitle } from "@yetric/ui";

   export default function App() {
     return (
       <Card className="max-w-sm mx-auto mt-16">
         <CardHeader>
           <CardTitle>Hello</CardTitle>
         </CardHeader>
         <CardContent>
           <Button>Get started</Button>
         </CardContent>
       </Card>
     );
   }
   ```

</Steps>

## Dark mode

Toggle the `dark` class on `<html>` to switch themes:

```ts
document.documentElement.classList.toggle("dark");
```

See the [dark mode guide](/theming/dark-mode/) for a full toggle implementation.

## Custom theme

To override color tokens, add a CSS file that overrides `:root` variables and import it after `@yetric/ui/styles`. See the [custom theme guide](/theming/custom-theme/).
